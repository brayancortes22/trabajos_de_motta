Equipo de Trabajo para el Desarrollo del Software de Bingo

1. Desarrollador Frontend : Daniel salazar vargas

Responsabilidades:

  • Diseñar e implementar la interfaz de usuario utilizando HTML, CSS y JavaScript.
  • Asegurarse de que la aplicación sea responsiva y accesible en diferentes dispositivos, especialmente en Android.
  • Integrar elementos visuales y de interacción que mejoren la experiencia del usuario.
  • Colaborar con el diseñador para implementar el prototipo creado en Figma.

Habilidades Requeridas:

  • Conocimiento sólido de HTML, CSS y JavaScript.
  • Experiencia en frameworks de frontend (opcional, como React o Vue.js).
  • Habilidades en diseño de interfaces y usabilidad.

2. Desarrollador Backend: Brayan Stid Cortes Lombana

Responsabilidades:

  • Diseñar y desarrollar la lógica del servidor y la API que gestionará las solicitudes del cliente.
  • Implementar la base de datos para almacenar información de usuarios, partidas y estadísticas.
  • Asegurar la seguridad y el cumplimiento normativo en la gestión de datos.
  • Colaborar con el desarrollador frontend para integrar el backend con la interfaz de usuario.

Habilidades Requeridas:

  • Conocimiento en lenguajes de programación de backend (como Node.js, Python o Java).
  • Experiencia en bases de datos (MySQL, MongoDB, etc.).
  • Comprensión de conceptos de seguridad en aplicaciones web.

3. Diseñador de Experiencia de Usuario (UX/UI): Paula Andrea Giraldo Losada

Responsabilidades:

  • Crear prototipos y maquetas de la interfaz de usuario utilizando herramientas como Figma.
  • Realizar investigaciones de usuarios y pruebas de usabilidad para mejorar la experiencia del juego.
  • Definir flujos de usuario y asegurarse de que la navegación sea intuitiva.
  • Colaborar con los desarrolladores para garantizar que el diseño se implemente correctamente.

Habilidades Requeridas:

  • Experiencia en diseño gráfico y herramientas de diseño (Figma, Adobe XD, etc.).
  • Conocimiento de principios de usabilidad y diseño centrado en el usuario.
  • Habilidades de comunicación para trabajar en equipo y presentar ideas.
